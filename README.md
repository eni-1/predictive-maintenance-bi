# Predictive Maintenance Dashboard


- Data Engineering: Automated Python ETL pipeline for cleaning, interpolating, and loading raw sensor data.

- Warehousing: Cloud-hosted PostgreSQL database using SQL Views to structure and aggregate data for reporting.

- Machine Learning: Scikit-Learn models (XGBoost & Random Forest) trained to predict failure probability and detect sensor anomalies.

- Reporting: Power BI dashboard utilizing DAX measures to visualize risk levels and operational costs.
